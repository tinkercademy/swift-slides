Unit 4B

Optional: Jokes

In this advanced unit, you'll make a Jokes app and implement structs and arrays, as well as create Alerts.

Unit 4C

Optional: Quiz v2

For this advanced version of the Quiz app, you'll use a Progress Indicator, Sheets, and create a Custom Button.

This is the quintessential "list app" with CRUD (create, read, update, delete) functionality. Learn about intermediate and advanced features such as Lists, modals, navigation views, and more.

Please note: This unit goes into quite a lot of detail on Swift and SwiftUI — please feel free to skip it if you don't have enough time!

Unit 6
