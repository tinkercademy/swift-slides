## Tip : Rounded Rectangle

- `RoundedRectangle` require a `cornerRadius` parameter to be set.

```swift[6]
import SwiftUI

struct ContentView: View {
    var body: some View {
        ZStack {
            RoundedRectangle(cornerRadius: 20)
        }
    }
}
```
