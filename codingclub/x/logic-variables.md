## Variables Summary

- A variable represents a value. The value can change, so the variable has a name for reference.  
- Declare a variable: `var variableName = 0`  
- Modify the variable:  
  ```swift
  myVariableName = 2  
  myVariableName = myVariableName + 1  
  ```  
- Constants: `let myConstant = 0`
