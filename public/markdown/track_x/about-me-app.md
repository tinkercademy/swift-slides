# Editing Tabs

---

## Changing Tab Labels

- Modify the `.tabItem` modifier to change the label text.  
- Example: Change `"Home"` to `"Your Name"`.

<img height="800" src="/markdown/track_x/assets/placeholder_editing_tab_labels.png">

## Adding Images to Tabs
